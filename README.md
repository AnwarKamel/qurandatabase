# Columns

* ``ID``
* ``NAME_AR`` (Name in Arabic)
* ``NAME_PRON_EN`` (Name pronunciation in English)
* ``CLASS`` (Mecca or Medina)
* ``VERSES_NUMBER``
* ``CONTENT``
